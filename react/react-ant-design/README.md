
It's a Yarn+Vite React project:
- Create React project
```
yarn create vite
```
First, choose project name, then choose react, and choose Javascript at the final step.
- Run project
```
yarn run dev
```
- Build project
```
yarn run build
```

## Dependencies
- **Ant Design**

```
yard add antd
```

- **React Browser Router**

```
yarn add react-router-dom
```
